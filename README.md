# Text Editor

A simple text editor built using Python and Tkinter.

## Features

* Create a new document
* Open existing text files
* Save files
* Save files with a new name (Save As)
* Cut text
* Copy text
* Paste text
* Dynamic menu state management
* Vertical scrollbar
* Custom application icon

## Note

Some menu options are currently under development and will be added in future updates:

* Undo
* Delete
* Go To
* Select All
* Zoom In / Zoom Out
* Restore Default Zoom
* Status Bar
* Print

## Requirements

* Python 3.8+ (tested on Python 3.13.5)

## Run Locally

```bash
git clone https://github.com/LokjitKundu/text_editor_tkinter.git

cd text_editor_tkinter

python run.py
```
